# Dashboard 🧭

A responsive dashboard interface built with HTML and Tailwind CSS. It focuses on layout, spacing, and clean visual hierarchy.

## Features ✨

- Responsive sidebar navigation 📱
- Top navigation with search and user area 🔎
- Project cards section 🗂️
- Announcements panel 📢
- Trending users panel 📈
- Font Awesome icons throughout 🎨

## Tech Stack 🛠️

- HTML5
- Tailwind CSS via CDN
- Font Awesome

## How to Run 🚀

1. Open the `7-dashboard` folder.
2. Open `index.html` in your browser.
3. Resize the browser to test the responsive layout.

## Notes 📝

This project is mostly focused on layout practice and dashboard styling. The content is intentionally simple so the structure can do the talking.
